# Шпион LAN (упрощенная настольная версия)

Локальная веб-игра по мотивам "Шпиона".

## Что реализовано

- Поле 7x7 (49 карт местоположений)
- У каждого игрока есть личная карта (одно из названий на поле)
- 4 действия:
  - `Убить` соседнюю карту
  - `Допрос` своей или соседней карты
  - `Сдвиг строки` (циклический)
  - `Сдвиг колонки` (циклический)
- Игроки ходят строго по очереди
- Победа при 3 очках
- Журнал событий и онлайн-синхронизация между клиентами через WebSocket

## Запуск

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Запустите сервер:

```bash
python app.py
```

3. Откройте у себя в браузере:

- `http://localhost:8000`

4. Чтобы другие устройства в вашей сети подключились к игре:

- Узнайте IP вашего ПК в локальной сети (например, `192.168.0.42`)
- Откройте с другого устройства:
  - `http://192.168.0.42:8000`

## Примечания

- Состояние игры хранится в памяти сервера.
- Если перезапустить сервер, партия начнется заново.
- На время разработки включен `reload=True` в `app.py`.
